2) create_cache.php
загружает или обновляет кэш для лоадов по диапазону выборки.

3) create_pproduct.php
создает pproduct, и прописывает его принадлежность в лоадере

4) create_product.php
создает товар и сразу загружает всю информацию по лоадерам

5) update_product.php
обновляет товары по выборке. из кэша или сразу с сайта создавая кэш
